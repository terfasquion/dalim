Exercises for OBJECTS and FUNCTIONS

Exercise #2

      Input a string.  

      Call a function that will return an array of objects.  
      One object for each letter with the key: "letter" and value: "A", 
      and another key for the amount of times found in the string.  
      Make sure all letter values in the object are uppercase, 
      and in alphabetical order. 

      Ex: 
      INPUT:  "Sabbah"
      OUTPUT:  
      [ 
        {letter: "A", amount: 2},  
        {letter: "B", amount: 2}, 
        {letter: "H", amount: 1}, 
        {letter: "S", amount: 1} 
      ]
